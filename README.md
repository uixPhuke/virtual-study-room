## Introduction

The **Virtual Study Room** enables users to:
- Study alone or with others in real time
- Stay accountable using timers and shared sessions
- Communicate via chat or video (optional)
- Maintain productivity using minimal and calming UI elements

This project is ideal for students, remote learners, and professionals who want a sense of shared focus without physical presence.
